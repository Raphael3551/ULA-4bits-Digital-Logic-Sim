# ULA de 4 Bits - Digital Logic Sim

Este projeto apresenta uma Unidade Lógica e Aritmética (ULA) completa, capaz de realizar operações de soma, subtração, multiplicação e divisão com 4 bits.

## Como baixar e executar
1. Baixe o simulador **Digital Logic Sim** [https://sebastian.itch.io/digital-logic-sim](https://sebastian.itch.io/digital-logic-sim).
2. Faça o download deste repositório.
3. Abra o simulador e vá em `File > Open`.
4. Selecione o arquivo `seu_projeto.circuit`.
